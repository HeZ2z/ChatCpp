# ChatCpp聊天系统

这是一个基于C++和WebSocket的聊天系统，支持多用户实时通信、历史记录和消息日志记录。

## 功能特点

- 多用户实时聊天
- 消息日志记录
- 命令行客户端
- 历史记录


## 编译

```bash
# 创建构建目录
mkdir -p build
cd build

# 配置和编译
cmake ..
make
```

## 运行

### 启动WebSocket服务器

```bash
# 在build目录下
./server [端口号]
```

如果不指定端口号，默认使用10808端口。

### 使用命令行客户端

```bash
# 在build目录下
./client <用户名> [服务器IP] [端口号]
```

例如：
```bash
./client alice 127.0.0.1 10808
```

如果未指定服务器IP和端口号，默认使用127.0.0.1:10808。

## 使用说明

### 命令行客户端

- 发送消息：直接输入内容并按回车
- 退出：输入 `\quit` 或 `\exit`

### 历史记录

登录时自动加载历史记录，以用户名 @ 消息 | 时间的形式保存。

例如：
```
alice @ 你好 | 2025-04-16 10:00:00
```

## 日志

所有消息都会被记录到服务器目录下的`chat_server.log`文件中。

## 故障排除

如果遇到连接问题：
1. 确保服务器正在运行
2. 检查防火墙设置，确保端口已开放
3. 确保客户端使用正确的服务器IP和端口
